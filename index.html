How it works

From phone call to token
in under 60 seconds

The entire experience happens over a simple phone call. No app, no internet, no waiting in line.

Features

Everything a clinic needs

Built for real-world use with reliability, multilingual support, and a live admin view.

📞
IVR Call Flow
Fully automated voice menu using Exotel. Patients navigate using their keypad — no smartphone needed.
🌐
Hindi & English
Bilingual support for both Hindi and English. Voice prompts and SMS messages adapt to the patient's choice.
🎫
Smart Token System
Auto-generates tokens per service (D-001 for Doctor, L-001 for Lab, P-001 for Pharmacy) with wait time estimates.
📱
SMS Notifications
Instant SMS confirmation with token number and expected wait time sent directly to the patient's phone.
📊
Live Admin Dashboard
Real-time queue monitor with service and status filters. Auto-refreshes every 10 seconds. Mark tokens as serving or done.
🔁
Status Check
Patients can call back anytime to hear their current position in queue — system recognises their number automatically.
